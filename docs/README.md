## Features:
* Compact Overlay
* Focus Mode
* Classic Mode
* Auto Save
* Dark Mode
* RichText suppprt
* Markdown Preview

## Shortcuts:
* Save - Ctrl + S
* Save As - Ctrl + Shift + S
* Open - Ctrl + O
* New Document - Ctrl + N
* Zoom In - Ctrl + +
* Zoom Out - Ctrl + -
* Reset Zoom - Ctrl + 0
* Superscript - Alt + + (rtf mode only)
* Subscript - Alt + - (rtf mode only)
* Classic Mode - Ctrl + Shift + 1
* Default Mode - Ctrl + Shift + 2
* Focus Mode - Ctrl + Shift + 3
* Pro Mode - Ctrl + F12
* Compact Overlay - Alt + Up
